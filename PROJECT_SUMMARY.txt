=== ИНСТРУКЦИЯ ПО РАЗВЕРТЫВАНИЮ ===

1. Создайте структуру папок (см. QUICKSTART.txt)
2. Переименуйте:
   - messenger_app_full.py → messenger_app.py
   - api_handler_full.py → api_handler.py
3. Скопируйте YAML конфигурации в соответствующие папки
4. Скопируйте providers.yml в папку providers/
5. Установите зависимости: pip install -r requirements.txt
6. Запустите: python messenger_app.py
