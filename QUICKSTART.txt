# БЫСТРЫЙ СТАРТ

## 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

## 2. Создание структуры папок
```bash
mkdir -p characters/jessica/photos
mkdir -p characters/mike/photos
mkdir -p characters/alex/photos
mkdir -p groups
mkdir -p chats
mkdir -p providers
```

## 3. Настройка файлов

### Скопируйте конфигурации:
- `character_jessica.yml` → `characters/jessica/character.yml`
- `character_mike.yml` → `characters/mike/character.yml`
- `character_alex.yml` → `characters/alex/character.yml`
- `work_group.yml` → `groups/work_group.yml`
- Ваш `providers.yml` → `providers/providers.yml`

### Переименуйте основные файлы:
- `messenger_app_full.py` → `messenger_app.py`
- `api_handler_full.py` → `api_handler.py`

## 4. Добавьте фотографии (опционально)
Поместите любые изображения в папки:
- `characters/jessica/photos/avatar.jpg`
- `characters/mike/photos/avatar.jpg`
- `characters/alex/photos/avatar.jpg`

## 5. Запуск
```bash
python messenger_app.py
```

## Примечания:
- Если у персонажей нет фото, будут использованы автогенерируемые аватарки
- Убедитесь, что в `providers.yml` указаны корректные API ключи
- Персонажи будут использовать разные провайдеры (Jessica → Claude, Mike → Gemini, Alex → OpenRouter)
